# 配置说明（CONFIGURATION）

所有配置在 Emby 后台 → 插件 → **Emby 飞书通知** 的 Simple UI 页面中设置，保存后立即生效，无需重启。
页面共 **8 个配置分组**：飞书连接、机器人安全校验、消息显示、播放通知、登录与用户、媒体库与用户行为、任务/Live TV/服务器、高级与诊断。下表按功能细分展开。带「高级」标记的项默认折叠在高级区域。

> **兼容性**：旧配置文件可直接加载。新字段缺失时使用下表默认值，播放通知外观保持不变（默认 `Custom` 详细程度）。插件 GUID、配置文件名、Webhook 均不改变。

## ① 飞书连接
| 配置 | 默认 | 范围/说明 |
| --- | --- | --- |
| 启用插件 | 关 | 总开关 |
| 飞书 Webhook 地址 | 空 | 必须 https:// |
| 请求超时（秒） | 10 | 3~60 |

## ② 机器人安全校验
需与飞书群机器人「安全设置」保持一致；插件不会自动修改飞书端配置。

| 配置 | 默认 | 说明 |
| --- | --- | --- |
| 启用自定义关键词 | 关 | 开启后每条消息（文本/卡片/回退）自动携带关键词，已含时不重复 |
| 自定义关键词 | 空 | 仅在启用时显示；须与飞书安全设置中的关键词完全一致 |
| 启用签名校验 | 关 | 与飞书「签名校验」对应 |
| 签名密钥 | 空 | `[IsPassword]` 遮蔽；输入新值并保存则更新，留空保持原值；非 Webhook Token |
| 签名密钥状态 | 未配置 | 只读，仅显示是否已配置，不暴露真实密钥 |

> 签名算法与安全细节见 [FEISHU-SECURITY.md](FEISHU-SECURITY.md)。

## 配置迁移

- 无 `MessageFormat` → `Text`；无 `MessageDetailLevel` → `Custom`（播放事件沿用旧字段开关，外观不变）。
- 现有播放开关、用户过滤、Webhook、超时/重试全部保留原值。
- 数值越界自动夹到合法范围（`ConfigMigrator`）。
- `ConfigSchemaVersion` 记录架构版本，供将来平滑迁移；不改变 GUID 与配置文件名。
